# Budget Tracker

A simple web application to track your income and expenses. Built with HTML, SASS, and JavaScript.

## How to Use

1. **Add Income:**
   - Enter a description (e.g., "Salary", "Bonus")
   - Enter the amount in dollars
   - Click "Add Income"
   - Your income will appear in the Income History with today's date

2. **Add Expense:**
   - Enter a description (e.g., "Rent", "Groceries")
   - Enter the amount in dollars
   - Click "Add Expense"
   - Your expense will appear in the Expense History with today's date

3. **View Summary:**
   - See your Total Income, Total Expenses, and Total Budget at a glance
   - The totals update automatically when you add new entries

4. **Clear All Data:**
   - Click the "Clear All Data" button to reset everything
   - You'll be asked to confirm before deleting

## Features

- Track multiple income sources and expenses
- See when each transaction was added (date tracking)
- Automatic budget calculation (Income - Expenses)
- Input validation (prevents empty fields and negative amounts)
- Clear all data with one click

## Technologies Used

- HTML5
- SASS/CSS
- JavaScript (ES6)
- Object-Oriented Programming

## Installation

1. Clone this repository
2. Open `index.html` in your browser
3. Start tracking your budget!
